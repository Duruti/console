# INFORMATION - MAIN RESOURCE                                                  #
################################################################################

The main resource for this PSX-program was the 
"Programmierkurs für die PSX, Teil 1" in "World of Playstation Magazin - 
Issue Nov. 1999 (German psx magazine)"

It's a very nice article about writing homebrew for the psx and delpoying an 
testing it on the real hardware via the xploder cheat module (unfortunately 
doesn't work without an old pc with lpt port and Win98 SE)

This example homebre runs on the ePSXe emulator and on the real hardware (burned
 as disc)
Therefore I've chosen the "Programmierkurs für die PSX, Teil 1"-code 
(by Bodo Hinüber) as a reference implementation for real PSX-hardware-compatible
 code.

Thanks to Bodo Hinüber and the team of the World of Playstation Magazin for 
preparing and writing an article about homebrew programming in 1999(!) in a 
common German psx magazine.
